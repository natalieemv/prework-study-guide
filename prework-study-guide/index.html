Git

  • git status:checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add-A saves changes for future commit (file must be saves on VS code as well)
  • git commit -m "with description of changes" will commit changes to branch
  • git pull origin main:will pull source code, pull request must be made before merging
  • git push origin (branch name):will push branch to main
